# 🔌 MCP Builder - MCP 服务器构建指南

## 概述

创建高质量的 MCP（Model Context Protocol）服务器，使 LLM 能够通过工具与外部服务交互。

**适用场景**：
- API 集成
- 数据库连接
- 外部服务接入
- 自定义工具开发
