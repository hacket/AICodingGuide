# 📝 Internal Comms - 内部沟通文档模板

## 概述

提供企业内部沟通文档的标准化模板和格式，包括状态报告、更新通知、FAQ 等。

**适用场景**：
- 项目状态报告
- 团队更新通知
- 事故报告
- FAQ 文档
- 公司通讯

## 🔧 使用方法

### 场景 1: 撰写项目状态报告

**示例需求**：
```
为 Q4 项目撰写月度状态报告
```

**包含内容**：
- 项目进展摘要
- 关键成就
- 风险和问题
- 下一步计划

---

### 场景 2: 编写事故报告

**示例需求**：
```
为系统故障编写事故报告
```

**包含内容**：
- 事故概述
- 影响范围
- 根本原因
- 补救措施
- 预防计划

---

### 场景 3: 创建团队 FAQ

**示例需求**：
```
为新产品功能创建 FAQ 文档
```

**包含内容**：
- 常见问题
- 详细解答
- 截图说明
- 联系方式
