# 🌐 Artifacts Builder - 复杂 HTML Artifacts 构建

## 概述

使用现代前端技术（React + Tailwind CSS + shadcn/ui）创建复杂的、多组件的 claude.ai HTML artifacts。

- 复杂交互式应用
- 多页面 Web 应用
- 数据仪表板
